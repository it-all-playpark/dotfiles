## Writing Guidelines

### For Decision-Makers

- Lead with business impact, not features
- Use percentage improvements over absolute numbers
- Include ROI indicators
- Keep sentences under 40 characters when possible
- Use tables for quick scanning

### Tone

- Professional but approachable
- Confident without exaggeration
- Evidence-based claims only
- No marketing superlatives ("revolutionary", "groundbreaking")

### Numbers to Include

| Type | Example |
|------|---------|
| Time savings | "月10時間 → 2時間（80%削減）" |
| Error reduction | "月3件のミス → 0件" |
| Cost impact | "年間XX万円相当の工数削減" |
| Quality metrics | "遵守率100%" |

### Common Challenge Patterns by Industry

| Industry | Common Challenges |
|----------|-------------------|
| 美容院/サロン | シフト作成負担、予約管理、スタッフ調整 |
| 小売 | 在庫管理、発注業務、売上分析 |
| 飲食 | オーダー管理、食材ロス、人員配置 |
| 製造 | 生産計画、品質管理、設備保全 |
| オフィス | 文書管理、承認フロー、情報共有 |

### Feature → Business Value Translation

| Technical Feature | Business Value Expression |
|-------------------|---------------------------|
| AI自動生成 | ワンクリックで完了、専門知識不要 |
| リアルタイム同期 | 最新情報を常に共有、二重入力不要 |
| ダッシュボード | 経営状況を一目で把握 |
| API連携 | 既存システムとシームレスに統合 |
| マルチテナント | 複数拠点を一元管理、拡大にも対応 |
